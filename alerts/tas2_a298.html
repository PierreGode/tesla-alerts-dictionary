TAS2_a298_vinNotLearned
Tesla Air Suspension 2 (TAS2) uses a VIN learn because the system learns the damper travel endstops to compensate for any differences between height calibrations and nonlinearity between vehicles.

A VIN has not been saved to the controller. Damping control will enter a backup state and endstop protection will not be active. This is expected for a new ECU that has just been installed into a vehicle, or on a vehicle where the ride height calibration has been cleared.

Set: The VIN learned to the controller does not match the VIN received on a car area network (CAN).

Clear: A VIN is saved by performing ride height calibration.

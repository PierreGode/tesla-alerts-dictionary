Pedal Monitor Warning: Torque Reversal.

Pedal Monitor Warning: Gear Shifter CAN node is missing.
The safety monitor sets this condition when CAN communication from Gear Shifter CAN node is considered irrational or missing.
Gear Shifter CAN node is considered not communicating on CAN for remainder of drive cycle.
Pedal Monitor Warning: Secondary CAN Integrity Check Failed.
This condiiton is set when CAN communication from an offending ECU is intermittent, or fails a communication integrity check on the primary CAN bus.
This alert will clear when CAN communication is re-established with the offending ECU.
No immediate impact, may lead to indicating an MIA condition for the offending ECU if sustained.
Pedal Monitor Warning: Absolute Torque.
This condiiton is set when CAN peripheral hardware indicates an overrun or transmission error on the primary CAN bus.
Pedal Monitor Warning: Inconsistent AEB state.
The safety monitor sets this condition when sensor or other ECU inputs into AEB are deemed implausible.
The safety monitor will disable automatic emergency braking (AEB) for the remainder of the drive cycle.

The safety monitor sets this condition when CAN communication from Cruise Control CAN node is considered irrational or missing.
Cruise Control CAN node is considered not communicating on CAN for remainder of drive cycle.

BMS_a016_Pack_Module_Id_Mismatch
The module IDs do not match the pack type set by the config block
Any module has all devices reporting invalid module IDs with correct CRCs, or any module has all devices reporting incorrect module ID CRCs (corrupt module ID data)
All modules have at least one device reporting a valid module ID with a correct CRC
May indicate an incompatible module has been installed for the pack type, which could lead to range loss due to CAC and SOC imbalance. If all modules are reported with invalid module IDs, this may indicate that the wrong config block has been written for the given pack hardware

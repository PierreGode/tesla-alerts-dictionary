CP_w029_doorClosedProxPilot
This particular fault is quite heavily debounced in the Charge Port (CP). It only sets when the following conditions have been met for over 3 seconds:
  • Door sensed as closed.
  • Valid pilot (LINE_CHARGE, FAST_CHARGE, or FAULTED) or proximity (LATCHED or UNLATCHED) is present.

If there is an issue with the door sensor that indicates the position as closed when it is opened, this alert may be triggered. Also, the alert can be triggered if the pilot or proximity state reported by the VI was stale / delayed.

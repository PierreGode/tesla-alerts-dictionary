EPAS_d002_KFC_NVRAM_MNG_READ
A read job for the Non-Volatile Random Access Memory (NVRAM) manager has failed. Both the main and redundant blocks could not be read from the NVRAM. This can be caused by a defective NVRAM, an incorrect Cyclic Redundancy Check (CRC), a misconfigured NVRAM, or the drivers not being initialized.

The resent conditions require a software restart, function inhibition reset, ignition change, and another function inhibition reset.

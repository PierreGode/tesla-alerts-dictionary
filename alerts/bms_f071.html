BMS_f071_SW_SM_TransCon_Not_Met
The battery management system (BMS) has detected that there is an issue with another system and has prevented the high voltage (HV) battery from being requested.

If this alert is present, the BMS state may not have matched the requested BMS state for a timeout period specific to both the present and requested state. The timeout period is initialized according to the present and requested state each time the requested state changes.

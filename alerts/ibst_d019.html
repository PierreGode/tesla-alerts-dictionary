IBST_d019_unsupportedHw
The used hardware revision / component does not match with the flashed software. This fault is used during development to handle software / hardware dependencies. The software compares the application-specific integrated circuit (ASIC) / Microcontroller revisions, the IDs read from the hardware to the revisions, and the IDs configured in the software. 

The presence of this alert indicates ?the revision or the ID of the ASIC (system / Valve / RFP ASIC) or the microcontroller does not fit with the software configuration.

PM_a060_onePedalDrivingInhibit
Pedal Monitor Warning: One Pedal Driving module inhibited.
The safety monitor sets this condition when torque production error is detected while One Pedal Driving module is active.
The safety monitor will disable the One Pedal Driving module for the remainder of the drive cycle.
Pedal Monitor Warning: Stability Control actuated in the wrong direction.
This condiiton is set when CAN communication from an offending ECU is intermittent, or fails a communication integrity check on the secondary CAN bus.
Pedal Monitor Warning: ESP CAN node is missing.
The safety monitor sets this condition when CAN communication from ESP is considered irrational or missing.
ESP is considered not communicating on CAN for remainder of drive cycle.
Pedal Monitor Fault: Torque Command Qualifier set to faulted
The safety monitor sets this condition torque production error is detected

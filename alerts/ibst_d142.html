IBST_d142_pdmHwAcsErr
Hardware error is reported by an underlying hardware driver while accessing the device for storing persistent data (reported for diagnostic purposes only, no system degradation). 

In the case of write access fails, the error is passed to the application using the NVM service. The application then performs the individual failure handling (e.g., system degradation and another failure is set). This additional failure entry only logs the root cause for hardware failure analysis.

EPAS_d003_KFC_NVRAM_MNG_WRITE
A write to the Non-Volatile Random Access Memory (NVRAM) (DataFlash) has failed. Both the main and redundant blocks could not be written to the NVRAM. This may be caused by a defective NVRAM, incorrect Cyclic Redundancy Check (CRC), misconfigured NVRAM, or the drivers were not initialized.

The reset conditions require a software restart, function inhibition reset, ignition change, and another function inhibition reset.

TAS2_a297_vinMismatch
Tesla Air Suspension 2 (TAS2) uses a VIN learn because the system learns the damper travel endstops to compensate for any differences between height calibrations and nonlinearity between vehicles. 

The VIN learned to the controller does not match the VIN received on a car area network (CAN). Learned endstops may be incorrect. Damping control will enter a backup state and endstop protection will not be active.

Set: The VIN learned to the controller does not match the VIN received on CAN.
Clear: A VIN is saved by performing ride height calibration.

Note: If using the normal air suspension calibration panel in TBX2 clears TAS2_a297_vinMismatch, perform another calibration using the "Air Suspension Calibration (Advanced)" Panel.

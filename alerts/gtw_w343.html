GTW_w343_frunkLatchFailure
This alert will appear if the status of the front trunk switch goes into an ambiguous state. Ambiguous state for the switch means the body controller (BCM - Model S and BCFRONT - Model X) sent a valid command to the front trunk latch but did not detected that the front trunk latch switch transitioned to OPEN state. A valid command means all preconditions were OK, and the module applied power to the latch.

??If this alert is present, Cruise Control will be disabled on Drivers Assistance System (DAS) vehicles (article 20365).

The alert will clear itself after a successful release cycle: if the primary latch switch transits from CLOSED to OPEN state within 200ms following a release request.

THC_w0148_refrigerantLow
The THC_w0148_refrigerantLow? warning was first released in FW 2017.24+

The THC_w0148_refrigerantLow alert shall be set if the following conditions are met for 60 seconds:
  1. THC is initialized 
  2. The ambient temperature sensor is not faulted
  3. The ambient temperature > 0 deg C
  4. Refrigerant discharge temp > 0 deg C
  5. There are no refrigerant discharge temperature alerts (THC_a0137_rfrgDisTemp or THC_a0138_rfrgDisTempRange)
  6. There are no refrigerant suction temperature alerts (THC_a0142_rfrgSuctPress) or THC_a0145_rfrgSuctPressRange)
  7. The refrigerant discharge pressure < 0.25 bar
 
The THC_w0148_refrigerantLow alert shall be cleared if the following conditions are met for 30 seconds:
  1. THC is initialized 
  2. The ambient temperature sensor is not faulted 
  3. The ambient temperature > 0 deg C 
  4. Refrigerant discharge temp > 0 deg C 
  5. There are no refrigerant discharge temperature alerts (THC_a0137_rfrgDisTemp or THC_a0138_rfrgDisTempRange) 
  6. There are no refrigerant suction temperature alerts (THC_a0142_rfrgSuctPress) or THC_a0145_rfrgSuctPressRange)
  7. The refrigerant discharge pressure > 1.5 bar?

Note: This warning will be customer facing in 2017.50.

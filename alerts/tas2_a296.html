TAS2_a296_chassisTypeMismatch
Tesla Air Suspension 2 (TAS2) uses a common PCBA/Hardware ID for Model S and Model X, but uses different software applications (.hex files / configs) between the two vehicle types. If this alert is active, the chassis type of the firmware on the controller does not match the chassis type received on a car area network (CAN). Damping control will enter a backup state and endstop protection will not be active.

Set: GTW_chassisType does not match the internal chassisType of the firmware on the controller.
Clear: GTW_chassisType matches the internal chassisType of the firmware on the controller.

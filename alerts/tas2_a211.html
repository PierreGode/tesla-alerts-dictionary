TAS2_a211_noRideHeightCalib
This alert indicates that the air suspension controller does not have stored ride height calibration. This is expected for a "virgin" board but should not be seen after the suspension ride height has been successfully calibrated. The alert checks for height sensor calibration status once per second.

Set: If height sensor calibration values are unavailable in memory, the alert is set.
Clear: If height sensor calibration values become available in memory, the alert is cleared.

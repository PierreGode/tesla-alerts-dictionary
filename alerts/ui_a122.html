UI_a122_AMDI2C0LostArbitration
AMD i2c Bus 0 lost arbitration during a transaction
Display status check failed due to datastream deserialization error.
The deserialiser check failed because I2C bus 2 failed to open or reading the deserialization chip id failed.
The I2C bus 2 has to initialize and the deserialization chip id has to be read successfully.
Touchscreen is blank, unresponsive or has limited functionality.

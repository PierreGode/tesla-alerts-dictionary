ESP_d059_aysPlaus
Lateral Acceleration Sensor: Plausibility against second signal failed. Ays plausibility fault.

The monitoring checks for a physically plausible lateral acceleration signal and consists of the following parts:
  • Monitoring during validity by building a reference lateral acceleration from all available sensor signals.
  • Steering angle.
  • Yaw rate.
  • Lateral acceleration.
  • Wheel speed sensors.

It is possible to test the lateral acceleration (Ay) signal on plausibility. If during stable vehicle behavior a lateral acceleration failure larger than approximately 2.5m/sï¿½ occurs, the vehicle dynamic controller will disregard the sensor information so a false vehicle dynamic intervention is prevented. A fault is recognized after 1.6s during model validity.

When observing, the recognition time depends on the amount of failure. The deviation of the measured signal from the model values is integrated when exceeding the allowed threshold. The shutdown threshold for the fault integral is 10m/s.

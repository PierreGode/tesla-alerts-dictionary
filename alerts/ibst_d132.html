IBST_d132_esmSafetyFlt
During a normal run, the error signal module (ESM) detects a hard failure in the configured ESM channels related to the uC core or the uC peripherals. A uC failure reported via the ESM occurred. A SoftReset was executed depending on the ESM Error and, the failure is stored after a SW reset. 
The ESM is a uC module in a uC safety layer, which handles severe uC core or uC peripheral failures and pulls error pin connected to external ASIC.
